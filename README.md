# 📊 SocialTrack Pro (SMAM)

## Overview

**SocialTrack Pro (Social Media Analytics Manager)** is a business intelligence and social media reporting application developed using **Python, Streamlit, Pandas, and Excel**. It is designed to eliminate the inefficiencies of manual social media reporting by providing a centralized platform where employees can submit their daily or weekly social media performance data, while managers can instantly access consolidated analytics, rankings, and reports.

Traditionally, organizations collect social media statistics manually and spend considerable time calculating total posts, views, and overall performance using spreadsheets. SocialTrack Pro automates this entire workflow by storing all employee submissions in an Excel database and generating real-time analytics without requiring any manual calculations.

---

## Problem Statement

Many organizations still rely on manually entering social media statistics into spreadsheets. At the end of the reporting period, managers must calculate totals, compare employee performance, and prepare reports manually. This process is:

* Time-consuming
* Error-prone
* Difficult to manage as data grows
* Inefficient for performance tracking
* Lacks real-time insights

SocialTrack Pro addresses these challenges by automating data collection, aggregation, and reporting through an interactive dashboard.

---

## Solution

SocialTrack Pro provides a simple yet powerful platform where employees enter their social media statistics, and the system automatically stores, organizes, and analyzes the data.

Instead of spending hours creating reports manually, managers receive an instant overview of team performance, platform statistics, employee rankings, and business insights.

---

# Key Features

## 📝 Data Entry Module

Employees can easily submit their social media performance through an intuitive interface.

The application collects:

* Employee Name
* Twitter Posts
* Twitter Views
* Instagram Posts
* Instagram Views
* Facebook Posts
* Facebook Views
* TikTok Posts
* TikTok Views

Every submission is automatically stored in an Excel database without overwriting previous records.

---

## 📊 Dashboard

The Dashboard provides an executive overview of social media performance.

It automatically calculates:

* Total Posts
* Total Views
* Total Employees
* Best Performing Platform

The dashboard also includes:

* Platform-wise summary
* Employee records table
* Real-time analytics
* Automatically updated reports

Managers can instantly monitor overall team performance without manually opening or editing Excel files.

---

## 🏆 Employee Leaderboard

The Employee Leaderboard automatically ranks employees according to their overall performance.

For every employee, the system calculates:

* Total Posts
* Total Views

Employees are then ranked from highest to lowest based on total views, with the top three performers highlighted using medal rankings:

🥇 First Place

🥈 Second Place

🥉 Third Place

This feature enables organizations to recognize top-performing team members and encourages healthy competition.

---

## ⚙️ Data Management

The Data Management module provides administrators with tools to maintain the application's data.

Current features include:

* Display total stored records
* Show active database file information
* Secure reset functionality
* Confirmation before deleting all records

This ensures that accidental deletion of company data is prevented while still allowing administrators to reset the system when required.
